# ui-ux-sambungtani
Aplikasi ini mendukung empat peran kunci: petani sebagai penjual, distributor sebagai pembeli, konten kreator sebagai promotor, dan agrikit sebagai penyedia alat serta bahan pertanian. Dengan dukungan AI, setiap transaksi dan interaksi dalam aplikasi menjadi lebih cepat, efisien, dan tepat sasaran.
